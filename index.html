G Ionian

By default this fretboard shows the G major scale (AKA G ionian). You can use this tool to display any mode of the major scale, in any key, by dragging the pattern on the fretboard to the left.

Instructions:

  • Drag left on the fretboard to move the root note (red dots) if you wish to change keys.
  • Drag left in the area below the fretboard to change modes. This will only move the white dots. As each white dot passes under a red dot (root note), the mode increases by one degree.
  • Protip: This site functions better with Firefox or Chrome, and will not function without Javascript enabled.
Remember! The diatonic modes, in order:
  1. Ionian (the Major scale)
  2. Dorian
  3. Phrygian
  4. Lydian
  5. Mixolydian
  6. Aeolian (the Minor scale)
  7. Locrian
